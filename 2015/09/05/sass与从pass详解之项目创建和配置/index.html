配置更改之后，我们的项目并没有变化，依然只有一个 sass 文件夹。现在我们来创建文件夹，一条命令就好：

我们可以在 sass 文件夹中再建三个 .scss 文件

_varible.scss：用来存放变量
_mixin.scss：用来放混合宏
index.scss：页面的主体scss文件

然后再在根目录建一个 index.html 文件。

到此，我们的项目基本就创建完了：

- .sass-cache
- css
- sass
  |-- _varible.scss
  |-- _mixin.scss
  |-- index.scss
- fonts
- images
- js
- index.html
- config.rb

注意：
应该有朋友发现 _varible.scss 和 _mixin.scss 两个文件名前面有下划线 _ ，这个下划线非常重要，加了下划线的scss文件就不会被编译成css文件。另外 .sass-cache 是编译缓存文件，大家不用管他。

现在可以把这个项目结构保存下来，以后要用，直接复制一份就好，然后再根据实际项目增添些文件就OK。
